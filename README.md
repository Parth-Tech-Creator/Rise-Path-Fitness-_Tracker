# 🏋️‍♂️ RisePath Fitness Tracker

A modern fitness tracking web application designed to help users monitor workouts, track progress, and stay consistent on their fitness journey.

---

## 🚀 Features

- 📊 Track daily workouts and activities  
- 🔥 Monitor fitness progress over time  
- 🗂️ Organized data display for easy understanding  
- 💻 Clean and user-friendly interface  
- ⚡ Fast and efficient performance  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Tools: Git, VS Code  

---

## ⚙️ Installation & Setup

1. Clone the repository:
```
git clone https://github.com/Parth-Tech-Creator/Rise-Path-Fitness-_Tracker.git
```

2. Navigate to project folder:
```
cd RisePath-Fitness-Tracker
```

3. Open `index.html` in browser
